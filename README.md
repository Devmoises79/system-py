# 🧠 System-Py — Exercício de Lógica em Python

📖 Sobre o Projeto

Este projeto consiste em um sistema básico desenvolvido em Python com foco em prática de lógica de programação.

A aplicação realiza interação via terminal, coletando dados do usuário e aplicando validações simples baseadas em condições.

# 🎯 Objetivos

- Praticar entrada e saída de dados em Python
- Trabalhar com estruturas condicionais (if/else)
- Manipular tipos de dados (strings e inteiros)
- Simular validação de acesso
- Reforçar lógica básica de programação

# ⚙️ Tecnologias Utilizadas
- Python 3

# 🧩 Funcionalidades

- 👤 Coleta de nome do usuário
- 🎂 Coleta de idade
- 🔢 Conversão de tipo de dados (int)
- ⚖️ Verificação de maioridade
- 🔐 Simulação de autorização de acesso

📌 Exemplo de Execução: 

```text
Olá, qual é o seu nome?
Moisés

Olá Moisés, quantos anos você tem?
20

Olá Moisés, você tem 20 anos.
Você é maior de idade. Usuário autorizado.
```

# 🧠 Aprendizados

Este projeto reforça conceitos fundamentais:

- Estruturas condicionais
- Entrada de dados pelo terminal
- Conversão de tipos
- Fluxo lógico de decisão
- Construção de sistemas básicos

# 🚀 Execução

```bash
python minisystem.py
```
